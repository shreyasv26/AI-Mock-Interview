AI Interview Prep Tool: Project README🌟 Project OverviewThe AI Interview Prep Tool is a full-stack application designed to help users prepare for technical interviews. The platform allows users to upload their resume, select a job role, and participate in a mock interview session powered by the Google Gemini API. It analyzes the user's resume, generates personalized questions, evaluates responses, provides detailed feedback, and tracks progress over time.💡 Key FeaturesUser Authentication: Secure registration and session management using JWT (JSON Web Tokens).Resume-Based Personalization: Upload and parse PDF/DOCX resumes to tailor interview questions based on user skills, projects, and experience.AI-Powered Interviews: Generates realistic, role-specific questions and provides sophisticated grading and feedback using the Gemini API.Progress Tracking: Stores and visualizes historical interview performance across various metrics.Learning Resources: Provides structured learning materials and resource links based on skill gaps identified in interviews.Modern Interface: Built with React and styled using Tailwind CSS for a responsive and modern user experience.🛠️ Tech StackComponentTechnologyDescriptionBackendPython (Flask)REST API server for logic, database interaction, and Gemini API calls.AI/LLMGoogle Gemini APIGenerates questions, scores answers, and provides feedback.DatabaseMySQLPersistent storage for users, sessions, and interview data.Resume ParsingspaCy, PyMuPDF, python-docxExtracts structured data (skills, experience, etc.) from resumes.FrontendReact (JSX), JavaScriptSingle Page Application (SPA) for the user interface.StylingTailwind CSSUtility-first CSS PrerequisitesPython 3.xMySQL ServerNode.js & npm/yarn (for the React Frontend)2. Backend Setup (Python)Clone the repository:Bashgit clone <your-repo-url>
cd <project-directory>
Create and activate a virtual environment:Bashpython -m venv venv
source venv/bin/activate
Install dependencies:Bashpip install -r requirements.txt
Install the spaCy NLP model:Bashpython -m spacy download en_core_web_sm
Configure Environment Variables:Create a file named .env in the root directory and fill in your credentials.Code snippet# --- AI API Keys ---
GEMINI_API_KEY=YOUR_GEMINI_API_KEY_HERE

# --- Authentication Secrets ---
# Used for Flask sessions and JWT token signing. MUST be strong and random.
SECRET_KEY=A_VERY_LONG_AND_RANDOM_SECRET_KEY
JWT_SECRET=A_VERY_LONG_AND_RANDOM_SECRET_KEY

# --- Database Connection (MySQL) ---
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=ai_interview
DB_USER=root
DB_PASSWORD=YOUR_MYSQL_PASSWORD
3. Database SetupLog into your MySQL server and create the database:SQLCREATE DATABASE ai_interview;
CRITICAL: Ensure all tables (users, resumes, sessions, questions, answers, progress, learn_section) are created with the exact schema provided in the section above.4. Frontend Setup (React)Assuming your frontend files (App.jsx, index.jsx) are in the root or a dedicated folder:Install Node.js dependencies:Bashnpm install
# or
yarn install
Check API URL: Ensure the BASE_URL in App.jsx points to your Flask server (default: http://127.0.0.1:5000).▶️ Running the ApplicationStart the Backend API (Terminal 1):Bashflask run
# OR
python app.py
The API should be available at http://127.0.0.1:5000.Start the Frontend Development Server (Terminal 2):Bashnpm start
# or
yarn start
The application should open in your browser, usually at http://localhost:3000.
